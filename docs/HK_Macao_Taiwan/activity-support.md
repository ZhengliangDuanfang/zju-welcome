# 活动和支持

## 社团和活动

- 港澳台学生交流协会（HMT）、学生粤语社、多个国际学生组织都会开展丰富的学生活动，提供大量港澳台学生专属的课程、体验、实习甚至旅游机会和资讯，千万不要错过！开学后会有相关机构组织老师和社团为港澳台新生举行专场分享，届时将向大家开放加入社团和学生群组的渠道，敬请留意消息通知。
- 浙江大学港澳台事务办公室负责处理港澳台学术与学生交流事务，每年都会组织开展港澳台劳动班、[港澳台社会实践团](http://www.ir.zju.edu.cn/2026/0529/c69795a3167834/page.psp)以及各类文化体验活动、组织港澳台学生中华文化知识大赛的校级初赛。参加活动不仅能认识更多港澳台同学、享受宝贵的实践机会，还能从中轻松获得二、三课堂学分。相关活动信息可在“浙大国际”微信公众号或[官方网站](http://www.ir.zju.edu.cn/69637/list.psp)找到。另外，港澳台同学也可以和其他同学一样，报名参加各学园、院系、社团组织举办的其他社会实践活动。

## 支持

- 学园、招生办、港澳台办事处以及港澳台协会等组织都会举办各种见面会、交流会、座谈会活动，帮助港澳台新生更快熟悉校园及丰富校园生活，欢迎大家踊跃参与。
- 除港澳台政府提供的资助和津贴，多家港澳台和本地企业也面向前往内地（大陆）高校就读的学生提供奖学金资助，相关资讯及申请方式一般可在港澳台本地教育局刊发的内地（大陆）高校升学指南中找到。另外，国家教育部设立港澳台华侨学生奖学金，可在学校通知后申请，与私企奖学金和政府资助不冲突。
- 如有任何生活和学习上的困难或疑问，首先可以找到辅导员求助。每个学园都设有港澳台学生专职负责人，必要时也可联系他们。当然，也欢迎和身边、CC98和港澳台的伙伴多多交流！
