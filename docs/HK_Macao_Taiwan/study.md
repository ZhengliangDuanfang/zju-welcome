# 学业

内地（大陆）的大学和港澳台高中学习风格差异很大，课业难度不算低，想要取得优异的成绩还是需要用功学习。能进入浙大的港澳台学生同样是高中阶段的优秀学生，只不过因为高中教学内容差异而更容易遇到衔接问题。大家大可以保持自信，勇敢跨过基础课程的衔接关口，打开大学学习的新一页。

港澳学生来到内地（大陆）求学，还可能会面对语言的巨大差异：港澳地区日常交流多以粤语进行、学术活动经常以英语进行，而这里则主要使用普通话。为了能在课堂上听讲、以及与老师和同学正常交流，建议有需要的港澳学生多加练习。除了在网上搜索自学资源外，有需要的同学可以考虑购买普通话考试教材，从而同时准备应考普通话考试，以便检视和证明在大学生活期间普通话水平的进步。

另一方面，同学可以多了解本专业基础名词的中英对照，这样既方便适应中文授课，也有利于提升英语能力，一举两得。由于这里一般使用简体字，简繁转换需要一定的过程，短时间内没办法调整过来也没关系，老师可以看懂，不会因为字形刁难学生。多数老师清楚港澳台学生的短板，只要积极主动，老师很愿意答疑解惑和提供额外的辅导。为了适应文化差异，港澳台同学可以多观察其他同学的语言习惯，以及适当浏览常用的社交网站、多看CC98论坛，促进了解和融入。

港澳台学生无需参与专业分流，可直接参照录取专业培养方案规划学业。除军训、思政和个别课程替代以外，所有课程修读和毕业要求与同专业其他学生一致。以下列出一些值得留意的部分：

## 军训和新生训练营

港澳台学生不参加军训，军需期间学校开设港澳台学生新生训练营，以帮助学生适应大学课程。原则上，所有港澳台新生都需要参加此课程。此课程不是正式课程，故不设签到或考核，也不计入成绩。

衔接课程面向所有港澳台学生，其实不一定能针对性地补充每人所需的知识点。而且，衔接课程更侧重于高中知识补充，未必与大一数理基础课程有非常紧密的呼应，更倾向于辅助作用。因此，建议有需要的港澳台学生发挥主观能动性，积极向衔接课程任课老师交流个人情况，甚至在新生军训时期自行购买大一教材来预习。

衔接班开展时间为军训期间，为期约2周。具体上课安排将在开学后由辅导员在港澳台学生群内发布。

## 思政课程
港澳台学生设有单独的思政课程修读要求，无需满足专业培养方案的规定，毕业所需学分相应减少。现行规定要求港澳台学生在思政、国情类课程中修读11学分。其中，所有学生必修《中国近现代史纲要（MARX1002G）》（3学分），港澳学生必修《军事理论（EDU2001G）》（2学分），台湾学生不修读《军事理论》。大一期间，学生还可参加线上课程《中国国情（ADMN1003G）》（1.5学分）。其余分数可在体育课之外的各类思政类课程、国情类课程中修读。国情类课程目录推荐参阅《港澳台新生手册》第16页，该手册收录了课程新代码。上述要求来源于《关于2020级及以后港澳台学生修读思政类和军体类课程的相关说明》。

思政课程如果讲授得当、内容精彩，则完全可以作为港澳台学生认识国情的平台，有助于港澳台学生培养国民身份认同和自豪感，明白国家方针政策，同时可以提供与内地（大陆）同学交流的机会。因此，如果条件允许，港澳台学生大可以选择修读一些由高水平教师任教的思政课程。另一方面，港澳台学生需要关注本科生院最新通知和自身修课情况，尽早达到本科生院规定的学分要求，以确保正常毕业。

思政课程会自动出现在每次选课前的预置课表里，港澳台学生可以按需考虑是否退选这些课程。如果确定不需要且没有意向修读某一门预置的思政课程，记得及时进行退选操作，以免在不知情下误选，尤其是《形势与政策》课程。

## 通识课程
由于中学课程和招生方法的差异，港澳台同学很可能遇到学业基础不同于主流的情况，因此追不上课程进度，这尤其常见于数理基础课程。特别地，一些在高中阶段完全没有学过物理的港澳同学，如果进入了必修大学物理课程的专业，则需要在短短一个学期内从头学习高中到大一程度的物理，造成不小的挑战。微积分和线性代数课程虽然是相当独立的知识体系，但实际教学中常会用到港澳地区高中阶段不一定强调的知识点，带来适应困难。

大一数理基础课程对日后专业学习有深远意义，很可能在往后的课程里需要用到，而且大一的学业适应情况也间接为本科四年的学习节奏打下基调，因此港澳台同学应当重视数理基础课程的衔接。首先，要有适当自信，不要因为基础问题而先入为主地害怕学习，应该把大学课程视为全新体系，尽可能跟上课堂进度，尤其是在微积分和线性代数的学习上。其次，要有弥补基础不足的意愿，尝试查阅高中资料，或者从更简单的教材入手。再者，要积极主动向外界求助，浙大不少数理基础课程的任课教师都尽职尽责、爱护学生，愿意帮助基础薄弱的同学。此外甚至可以考虑以有偿形式寻找补习帮助。

基于对以上问题的考虑，学校允许港澳台学生选修《微积分基础（MATH1131F、MATH1132F）》、《大学物理基础（PHY1001F、PHY2001F）》、《线性代数导论（MATH1231F）》等几门面向少数民族、港澳台学生及留学生开设的基础课程。修读上述课程可相应免修（即替换）《微积分甲》、《微积分乙》等同类型课程。需要注意，此类课程一般采用班级排名四级制评分，对应最高可获得绩点有所降低。此类课程一般允许免听，即可以通过补选与其他课程形成冲突选课，相对而言比较灵活，但需提前征得任课老师同意。如有意向选择此类课程必须单独咨询任课教师（可在选课阶段点击课程详情，找到教师的联系方式），并与选课组（88206187）确认。上述内容仅供参考。

## 成绩
- 我校课程一般由平时成绩和期末考试成绩组成，平时成绩一般包含期中考试、随堂测试、到课点名、作业和展示、发言等。部分通识课程会限制90分以上的人数，也有些课程采用班级排名赋分制，划定不高于一定分数的学生比例。大型通识课程期末考试一般采用双及格制，卷面低于一定分数同样视为不及格（挂科）。
- 部分大型通识课程设有补考，在下个学期开学时举行，挂科后可以选择参加。补考设有卷面最低分60分，只要超过此分数，算补考通过，分数计60分，否则为0。此纪录会和挂科的记录一起显示在成绩单上，类似于修读两次（且都是低分）。因此如果时间允许，建议重修或跟班补考。
- 必须修读并通过所有培养方案中规定的课程（或满足所有替代要求）才可以毕业。
- 我校出国成绩单汇报为4.3分制，与平时教务网使用的5.0分制有所不同，具体可参阅《出国成绩单平均学分绩点计算方法（2022级起）》。
- 出国成绩单中，相同课程号课程绩点取最高者参与计算，但保留每次记录并作标记。
