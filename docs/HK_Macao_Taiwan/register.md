# 报到

和所有本科新生一样，港澳台学生会收到相同的录取通知书和注册指引，按照流程完成网上注册和缴费即可。可以留意以下几点：

- 学校系统中的预留电话为各招生项目的考生信息页直接以文本录入，港澳台电话号码很可能格式错误，务必在激活通行证阶段维护一个可以使用的手机号，最好是中国大陆号码。
- 如果使用港澳台手机注册钉钉，要在办理境内手机号后及时更换（同时更换通行证信息和钉钉），有些依托钉钉平台绑定手机号的服务不具备向港澳台手机发送验证短信的能力。
- 无需缴纳军训服装费用，如果误缴学校会在一段时间后联系退费。
- 注册材料包括录取通知书、通行证或居住证。个别有普通高中就读经历的，可能还有学生/考生档案，可以自由选择是否提交给学校。也可携带其他重要文件备用（例如出生证明、疫苗针卡等），惟务必妥善保存。
- 由于港澳台学生通常会获得多份录取，不能保证学生选择入读，故寝室安排将在到校报到登记时确定。港澳台同学需要先在学园注册点签到，再到白沙综合楼宿舍服务中心办理宿舍登记。
- 从加入学校钉钉组织开始，学园会陆续组建港澳台学生群组，发布港澳台学生的相关安排和指引。报到前后有任何事项或疑问可在群组内直接联系负责老师或辅导员。
