# 证卡办理与使用

大家来到内地（大陆），想必会注意到证件方面差异，这可能是同学们适应这边学习生活首先要注意的一点。下面提示几点可以留意的基础事项，主要是各类证卡的办理和使用：

## 身份证件

在内地（大陆），在所有场景都获承认的有效身份证件是港澳/台湾居民来往内地（大陆）通行证（通行证/回乡证/台胞证）以及港澳/台湾居民居住证（居住证），二者可独立使用。前者是所有来到内地（大陆）的港澳台人士所持有，后者有一定办理条件。居住证基本相当于居民身份证，可以在几乎所有支持使用身份证的场景当作身份证使用（办理手机号等身份敏感业务除外）。部分需要登记身份的场合，例如献血站点、一些博物馆、小程序，可能只能使用居民身份证来登记，持有通行证的港澳台人员必须寻求人工服务帮助。港澳台地区的身份证则一般不使用。通常，使用居住证较为方便。

根据各计划设置的不同，港澳台同学的招生系统预登记证件可能是身份证、通行证或居住证的其中一种，而在校学生相关系统（经统一身份认证）用户信息已自动设置为通行证或居住证，密码设置为zju2026XXXX，后四位为对应证件尾数，但也不排除有录入错误的情况。如果系统错误预留了身份证，需要在线上注册开始后的上班时间电话联系统一认证后台（88981244）维护通行证或居住证信息，如果密码错误，可持校园卡和身份证件前往东区食堂校园卡服务部重置密码。后续获得的学生证、校园卡（及密码）、联名银行卡、考试报名等，都会关联此证件。因此，建议以学籍信息的证件作为主证件办理未来的各种实名业务及参加考试，可以省却不必要的麻烦。

### 居住证的办理

根据规定，在杭州居住半年以上的港澳台居民，可到居住地辖区派出所户籍窗口申请办理居住证。对于我校住宿的港澳台学生，可参阅以下办理流程：

居住满半年后，携带：

1. 有效期内的通行证；
2. 学生证；
3. 在读证明（行政服务办事大厅本科生院窗口开具）；
4. 住宿登记证明（紫金港校区白沙综合楼一楼学生公寓综合服务大厅开具，如住在留学生公寓，则在留学生公寓办理），

前往紫金港校区东4教学楼杭州市公安局西湖区分局紫金港综合服务大厅（求是派出所）办理。

当然，如果在本地有家庭固定住所或自己校外租房的（或单纯不想用宿舍作为地址），也可以持房产证或租房合同连同派出所或社区出具的居住登记表或证明书等相关材料到居住地户籍窗口办理。

新办理居住证不影响学信网和浙大学籍系统已绑定的证件号码，学籍系统只承认录取入学时的证件。因此，除非录取入学时使用居住证，否则一般不会出现以居住证号码开具校内证明、报考外语四六级考试等情况。

### 证件更新及出入境

如果是首次入境，根据《中华人民共和国出境入境管理法》，港澳台居民在旅馆以外的住所居住的，必须在入住后24小时内办理境外人员临时住宿登记。港澳台学生可在获取宿舍居住证明或校外租房合同后前往国家移民管理局“移民局12367”平台自助办理登记。现时，此政策通常不要求居住在学校宿舍的学生自行办理，但第一次入境的同学和校外住宿的同学需按要求办理。理论上，每次从港澳台地区或外国进入内地（大陆）后，都需要重新办理一次。如果仅是离开杭州市或浙江省范围，而没有离开境内范围，则不需要重新办理。

紫金港校区公安服务大厅可以受理居住证和通行证的补换发业务，该办事处配有现场免费摄像及邮递取证服务，如有疑问，可以咨询87281369。香港驻内地经济贸易办事处可以办理香港护照补换发业务；澳门护照、台湾护照可邮递（代）办理换发业务。

如遗失港澳身份证或台湾护照而急需返回，可直接凭通行证（回乡证/台胞证）通过两地边检，再进行申报和补办手续。如遗失通行证而急需返回，需先到派出所报失并取得回执，再前往公安局出入境接待大厅或管理处办理一次有效通行证出境。

## 电话号码

- 取得中国大陆电话号码至关重要。港澳台学生可以凭身份证件（通行证、居住证）在任何电讯营业厅申请开台，也可在港澳台当地办理大中华一卡多号业务。不过，各种线上办卡、学生优惠等活动渠道普遍不支持港澳台证件认证，如想参与活动，须由持有居民身份证的朋友代为申请并过户的方式方可享受优惠。
- 持有或使用他人实名的电话号码有法律风险，且无法通过一网通查核验，会对日后办理实名业务带来麻烦。
- 一般而言，只有使用港澳台地区电话卡漫游时才可以连接港澳台地区网络。
- 中国大陆的区号是+86，杭州的区号是0571，跨城拨打座机电话要加区号，国际电话要加0086和区号。例如，用港澳台手机拨打求是学院电话可拨0086(0571)88981791，拨打私人手机号可拨008617817561590。
- 提醒大家不要轻信报到期间校园内的手机电话卡流量卡推销，此等推销不是学校营业厅的授权业务，也不是学校组织的合作产品。开办此类电话卡恐有财产损失和使用不稳定的风险，请务必亲临营业厅柜台开办。（他人线上代为办理的，一定要从正规入口申请，校内推销人员提供的二维码或链接与学校营业厅或学校组织无任何关联）

## 寻医问药

港澳台学生可以选择参加学生医保，相关收费和保障待遇和其余学生一致，详见《关于2026级新生参加杭州市大学生医保有关事项的通知》。需要注意的是，港澳台学生在线上登记缴费完成后，还需要持证件到西湖区政务服务中心办理医保开设手续，并当场取得实体卡。电子医保码无法获取或已获取但刷码使用不成功是正常现象，因此请随身携带实体卡，使用实体卡就医。

目前发行的医保卡实体卡为多功能卡片，可激活银行账户功能和交通卡功能。办理卡片时可以自由选择银行，开通金融功能和开立新户口无异，如非有使用需要，可以不激活。详情可以咨询校医保办（88981591）。地址：紫金港纳米楼行政服务办事大厅115室21号窗口。

港澳台学生须参加新生统一体检，一般在军训期间开展，现场缴费。在境内体检和就医一般无需提供过往的健康材料，但如果需要打疫苗，必须携带港澳台地区的针卡。

港澳台地区普通医疗保险计划一般不适用于中国大陆，全球保障或已包含的保障产品，一般要求于指定名单医院或三甲医院就医，并开具病例。请务必关注保险保障范围及理赔条例，以免耽误治疗和理赔。

中国大陆的医疗机构能提供的药物可能与港澳台地区不同。港澳台同学若有需要，在遵守出入境法规限制、符合卫生法规管制的前提下，可能会带同合理自用范围内的药物入境。请务必自行了解两地对具体药物各自的管制办法，尤其是海关规定！

浙大学生普遍采用的就诊方式是前往校园附近的医院，尤其是浙大校医院。如果想了解更多医院信息和就诊攻略，请阅读“生活-就医服务”章节。

## 银行户口

港澳台学生不参与开学时的学校联名银行卡统一办理，需电话预约（87814191）并持身份证件及学生证到中国银行紫金港支行办理。开办此卡必须使用和学校信息相同的主身份证件。校园联名卡不是必须办理。

开设其他银行户口时两种证件（通行证、居住证）都可使用。银行普遍支持通行证开户，居住证开户则一般要求配合通行证使用，且在同一银行一般只允许用一个证件作为主身份信息。需要留意，两种证件实名的户口在绑定其他平台或作实名验证时认可度和作用可能不同，例如微信绑定银行卡普遍支持通行证，而居住证支持度较少；但在一些仅支持身份证验证场景下，居住证办理的卡又可视为身份证通过验证。可以两种身份信息的户口都开设一张（不同银行）。推荐在办理同时要求开通网上银行。

开设中国大陆银行户口需要携带的文件一般包括：通行证、学生证、本地电话号码。有些时候，可能还要求提供在读证明、地址证明、港澳台身份证等。需要注意，分行一般会检查电话号码实名信息与本人是否一致。如果不一致，必须填写单据保证号码所有人为直系亲属，有些还要求提供证明。因此，如果号码为父母办理，请先准备好相关证明，例如载有户主姓名的发票及其身份证明文件副本。最好在成年后办理。

港澳台用户开户手续一般较多，请做好需要等很久或跑两次的准备。后续使用中，偶尔可能出现风控冻结卡片等情形，无需慌张，带齐证件到分行处理即可。境内银行的对私（个人）业务一般只在工作日提供，个别市级支行可能在周末营业。
