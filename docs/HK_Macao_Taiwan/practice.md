# 课余活动与实践实习

## 活动
港澳台学生完全可以像其他新生那样参加社团组织，以及报名参加各种课余活动，从而与内地（大陆）的同学互相认识和交流。如果想认识同样来自港澳台地区的朋友，也以参加浙大港澳台事务办公室举行的活动，例如见面会、劳动体验、实地参访等，相关信息会在[官方网站](http://www.ir.zju.edu.cn/69637/list.psp)更新。

## 社会实践
浙大每年寒暑假都会举行大量的社会实践活动，组织学生到祖国各地进行公益服务、参访考察等实地学习。这些活动有利于学生认识国情和接触社会，本就值得港澳台学生参加；而且，每一位学生必须至少参加一次社会实践才能毕业，因此更是必须参与了。浙大港澳台事务办公室每年举办[赴云南景东社会实践活动](http://www.ir.zju.edu.cn/2026/0529/c69795a3167834/page.psp)，内容包括短期支教和乡村考察，港澳台同学可以关注通知并积极报名。另外，港澳台同学也可以像其他同学那样，报名参加各学园、院系、社团组织举办的其他社会实践活动。

## 就业实习
每年，中国科协和教育部都会共同主办港澳台大学生暑期实习活动，而港澳台地区的官方或民间机构也会各自发起实习计划，港澳台学生可以在网上自行搜索相关信息。

港澳台学生也可以像其他同学那样，自行联系社会上的工作单位，寻求实习机会。持有通行证就能在内地（大陆）合法从事有偿劳动。
