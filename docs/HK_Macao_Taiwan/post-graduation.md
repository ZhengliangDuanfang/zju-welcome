# 升学和就业

内地（大陆）盛行深造风气，不少学生在本科低年级已经关注在本科毕业后如何进一步攻读研究生学位，而港澳台同学也很可能对此感兴趣。港澳台学生也可以考虑在本地就业，实现“把个人理想融入国家发展伟业” 、“让个人奋斗汇入国家发展洪流”。

## 港澳台研究生招生

港澳台同学如果有意在内地（大陆）攻读硕士研究生，可以参加“面向港澳台地区招收研究生”安排。这个招生安排仅面向港澳台人员，因此竞争压力较小，但考试内容、形式、时间都存在相当大的变数。因此，如果有意报考，务必密切关注[官方网站](https://www.gatzs.com.cn/)和相关报考单位的通知。要注意的是，这种招生安排要求考生必须获得拟报考专业的两位专家（副教授以上职称）书面推荐，因此有意深造的港澳台同学可以在大学期间积极与老师交流，尤其多认识拟报考专业的老师。近年来越来越多微信公众号提供港澳台考研相关信息，同学们可以谨慎阅读和参考。

北京大学、清华大学、暨南大学、华侨大学有单独的招收港澳台硕士生方法，独立于上述招生安排之外，港澳台同学可自行关注官方招生信息。

## 全国考研

港澳台同学可以像其他同学那样，参加全国硕士研究生招生考试，也就是我们常说的考研。

## 保研

教育部2019年文件指出“原则上在内地（大陆）就读的港澳台优秀应届本科生均可按照有关规定申请推免攻读硕士研究生”。除留在本地升学外，港澳台同学当然也可以选择回到港澳台地区深造，或是出国留学。

## 实习

每年，中国科协和教育部都会共同主办港澳台大学生暑期实习活动，而港澳台地区的官方或民间机构也会各自发起实习计划，港澳台学生可以在网上自行搜索相关信息，或在港澳台相关群组中获得最新消息。港澳台学生也可以像其他同学那样，自行联系社会上的工作单位，寻求实习机会。

## 就业

如果选择在内地（大陆）就业，港澳台同学要注意工作单位区分为面向应届高校毕业生的招聘（校招）和面向社会人员的招聘（社招），前者又主要在秋季和春季两个时间点进行（分别称为春招和秋招），这些安排与港澳台地区不相同。持有通行证就能在内地（大陆）合法从事有偿劳动。

港澳台人员还可以用通行证或居住证，在杭州报考内地（大陆）的职业资格考试，例如教师资格考试等，港澳台同学可自行关注考务通知。

广东省部分城市曾推行定向招录港澳籍公务员，港澳同学可自行关注相关政策动向，留意就业要求。
