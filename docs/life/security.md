# 校园安全与保卫处

初入大学校园的同学们，在享受自由和多彩大学生活的同时，也需要保护自身安全！

**浙江大学安全保卫处**（简称“保卫处”）是维护校园安全秩序、保障师生人身财产安全的核心部门，也是每一位浙大学子在校园内遇到安全问题时应首先想到的求助对象。

## 各校区 24 小时报警求助电话

遇紧急情况，请拨打所在校区保卫处电话，保卫处 24 小时接警，比拨打外部报警电话响应更快：

| 校区 | 报警电话 |
|---|---|
| **五校区通拨** | **8 1897 110** |
| 紫金港校区 | 8820 6110 |
| 玉泉校区 | 8795 1110 |
| 西溪校区 | 8827 3110 |
| 华家池校区 | 8697 1110 |
| 之江校区 | 8659 2777 |
| 国际校区（海宁） | 8757 2116 / 8757 2119 |

紫金港校区校园"110"接警服务室位于**东四教学楼一楼**，邮箱：zjubw@zju.edu.cn。

如遇紧急情况也可直接拨打：火警 **119**、匪警 **110**、医疗急救 **120**、交通事故 **122**、反诈专线 **96110**。


## 保卫处服务项目

保卫处不仅负责接警处警，还提供以下日常服务：

| 服务项目 | 说明 |
|---|---|
| 失物招领 | 可前往校园 110 接警服务室登记查询 |
| 监控查询 | 如有物品遗失或遭遇不法侵害，可申请调取校园监控 |
| 户口迁移 | 新生户口迁入可向所在院系提交材料，由各校区户口管理办公室办理 |
| 无人机报备 | 校内飞行无人机需向保卫处进行报备 |
| 大型活动安保 | 社团活动、大型集会等可协调保卫处安排安全保障 |
