# 常用网站

海宁校区同时会用到国际校区和浙江大学校级系统。作业、通知和办事入口并不集中，建议至少收藏 **myZJU、Blackboard/Canvas、PeopleSoft、综合服务网和RVPN**。部分校内网站在校外需要先连接RVPN，具体使用哪个教学平台以学院和任课教师通知为准。

## 官网

| 网站 | 主要内容 | 适合查找 |
| --- | --- | --- |
| [浙江大学国际联合学院（海宁国际校区）](https://www.intl.zju.edu.cn/) | 海宁国际校区官方网站 | 校区新闻、通知、机构与办事信息 |
| [ZJE](https://zje.intl.zju.edu.cn/) | 浙江大学爱丁堡大学联合学院官方网站 | ZJE新闻、师资与教学信息 |
| [ZJUI](https://zjui.intl.zju.edu.cn/) | 浙江大学伊利诺伊大学厄巴纳香槟校区联合学院官方网站 | ZJUI新闻、师资与教学信息 |
| [浙江大学](https://www.zju.edu.cn/) | 浙江大学官方网站 | 学校新闻、机构与校级信息 |

## 教学类

| 网站 | 主要用途 | 使用提示 |
| --- | --- | --- |
| [Blackboard](https://learn.intl.zju.edu.cn/) | 课程通知、课件、作业、考试与成绩 | ZJE常用，以任课教师要求为准 |
| [PeopleSoft](https://scrsprd.zju.edu.cn/) | 选课，查询课表、考试、成绩和排名 | 选课和查成绩时高频使用 |

| [本科教学管理信息服务平台](http://zdbk.zju.edu.cn/) | 浙江大学校级选课、课表、考试、成绩及教务申请 | 校级课程或教务事项按通知使用 |
| [学在浙大](http://course.zju.edu.cn/) | 浙江大学线上课程学习平台 | 部分校级课程会在此发布资料和作业 |
| [浙江大学本科生院](https://ugrs.zju.edu.cn/) | 本科教学通知、培养方案与交流项目信息 | 查询校级教学政策和通知 |
